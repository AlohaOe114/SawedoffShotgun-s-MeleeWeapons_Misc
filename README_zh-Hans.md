# 描述

 一个 100% XML 的 Rimworld Mod，旨在添加一些可能适合现实生活和 Vanilla/CE 游戏环境的新战斗类机械族。
 本Mod具体内容的"关键词"：
  “近战“ 或 ”反蜈蚣”

 "Waghhh!"
