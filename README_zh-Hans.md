# 描述

 一个 100% XML 的 Rimworld Mod，旨在添加一些可能适合现实生活和 Vanilla/CE 游戏环境的新近战武器。
 本Mod具体内容的"关键词"：
  “近战“

 "近战武器，小子" 

# 兼容性:
 游戏中途加入-安全    
 原版Core兼容    
 
 ==**!!CombatExtended(CE) Compatible/CE兼容!!**==   

 CharacterEditor(CE) Compatible   
 CheatEngine(CE) may Compatible   

# 特性：

 #### Vanilla，但有 CE 补丁：
 - 刽子手 (近战; 反蜈蚣):    
 一种携带大型动力剑的巨型机械族
